## 🎨 设计规范

### 主题色使用

```css
--primary: #4C12A1;        /* 博士紫 - 主色 */
--social: #C964CF;         /* 社交紫 - 渐变辅助 */
--manage: #FC4C02;         /* 管理橙 - 通知/警示 */
--thinking: #FFA300;       /* 思维黄 - 学习相关 */
--understanding: #EF4A81;  /* 理解粉 - 咨询相关 */
--integration: #2DCCD3;    /* 感统蓝 - 打卡相关 */
```

### 颜色应用策略

| 功能模块 | 主色调 | 应用场景 |
|---------|--------|---------|
| 全局 | 博士紫 + 社交紫 | 主按钮、导航栏选中状态、渐变背景 |
| 咨询安排 | 理解粉 | 日历标记、咨询卡片 |
| 服务打卡 | 感统蓝 | 打卡按钮、状态标识 |
| 学习资料 | 思维黄 | 学习入口、进度条 |
| 通知/警示 | 管理橙 | 通知圆点、重要提示 |

### 交互规范

1. **卡片点击反馈：** `transform: scale(0.98)` + `opacity: 0.8`
2. **圆角：** 统一使用 `12px` 或 `8px`
3. **阴影：** `0 2px 8px rgba(0,0,0,0.05)`
4. **过渡动画：** `transition: all 0.3s`
