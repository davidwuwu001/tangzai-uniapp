## 🐛 常见问题排查

### 问题1：底部导航栏不显示或显示异常

**原因：** TabBar图标文件不存在

**解决：** 
已使用纯文字TabBar（无图标），应该正常显示

**检查：**
打开 `pages.json`，确认tabBar配置：
```json
"tabBar": {
  "list": [
    { "pagePath": "pages/teaching/teaching", "text": "教研" },
    { "pagePath": "pages/service/service", "text": "服务" },
    { "pagePath": "pages/mine/mine", "text": "我的" }
  ]
}
```

### 问题2：页面显示空白

**可能原因：**
- 编译失败
- 页面路径错误

**排查步骤：**
1. 查看HBuilderX控制台是否有错误
2. 查看微信开发者工具控制台
3. 确认pages.json配置正确

### 问题3：点击Tab没有反应

**可能原因：**
- pages.json中的pagePath路径错误

**检查：**
确认这些文件存在：
- `pages/teaching/teaching.vue`
- `pages/service/service.vue`
- `pages/mine/mine.vue`

### 问题4：搜索功能不工作

**排查：**
1. 打开控制台
2. 输入关键词后按回车
3. 查看是否有日志："搜索关键词: XXX"
4. 检查filteredCards计算属性
