## ⚠️ 注意事项

### TabBar图标缺失
由于当前还没有准备图标文件，TabBar在运行时可能显示异常。

**两种解决方案：**

**方案1：临时方案 - 仅使用文字**
修改 `pages.json`，删除 iconPath 配置：
```json
"tabBar": {
  "list": [
    { "pagePath": "pages/teaching/teaching", "text": "教研" },
    { "pagePath": "pages/service/service", "text": "服务" },
    { "pagePath": "pages/mine/mine", "text": "我的" }
  ]
}
```

**方案2：添加图标文件**
1. 创建目录 `static/tabbar/`
2. 准备6个图标（每个Tab 2个状态）：
   - teaching.png (未选中) + teaching-active.png (选中)
   - service.png (未选中) + service-active.png (选中)
   - mine.png (未选中) + mine-active.png (选中)
3. 图标尺寸建议：81px × 81px
4. 格式：PNG，带透明背景
